**UniDeal**

**Build Order · v3 · Antigravity Edition**

LPU Campus Marketplace · 6 Phases · Security gate after every phase

|     |     |
| --- | --- |
| **Stack** | Next.js 14 App Router · TypeScript · MongoDB Atlas · Firebase Auth · Vercel |
| **Auth** | Firebase Email/Password — any email allowed — JWT stored in HTTP-only cookies |
| **Roles** | user (default) · admin (moderates listings, bans users) · superadmin (full control) |
| **Default mode** | Listing approval: Automatic (configurable to Manual or AI-gated by superadmin) |
| **Images** | Cloudinary — max 6 per listing, 5MB each, 15MB total |
| **Contact** | WhatsApp deep link (waLink only) — phone NEVER in response or DOM |
| **Rate limit** | 20 contact requests per user per day via Upstash Redis |
| **AI moderation** | Gemini checks title + description + category match before approval |
| **Expiry** | Listings expire after 60 days — cron runs daily at 02:00 |
| **Format** | Each task = one Antigravity prompt. Copy verbatim. Verify before next task. |
